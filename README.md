# 전투 시스템 개요

## 주요 기능

플레이어 이동: 캐릭터의 이동 및 회전

콤보 액션: 연속 공격 액션

활 쏘기: 원거리 무기 사용

시점 변경: 1인칭/3인칭 시점 변경

시점 락온: 특정 적 캐릭터를 타겟으로 고정

## 모듈화

### 주요 모듈

캐릭터 모듈 (ThirdPerson)

캐릭터 이동 및 조작 기능

타겟 잠금 및 상태 변경 기능

무기 및 방어구 장착 기능

스태미나 및 체력 관리 기능

전투 모듈 (CombatSystem)

전투 시스템 (근접 및 원거리 공격)

적 캐릭터 스폰 및 웨이브 관리

게임 종료 처리 기능
